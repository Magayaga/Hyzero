# Get Started on Hyzero

## Usage

* Create a `.hyzero` file as `test.hyzero`

  ```
  print("Hello, World!")
  print("Welcome, Hyzero!")
  ```

* Run

  Either run from the interpreter directly:

  ```
  python hyzero.py test.hyzero
  ```

  Or run in an interactive shell:

  ```
  python shell.py
  run("test.hyzero")
  ```
* Output

  ```
  Hello, World!
  Welcome, Hyzero!
  ```
